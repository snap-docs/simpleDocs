# Pilot User Guide

## Sign in

1. Launch `Start-CodeExplainer.bat`
2. Enter the redeem code you received
3. Wait for the tray icon to show the app is ready

## Use the app

1. Highlight code, terminal output, or technical text
2. Press the configured hotkey
3. Read the overlay response
4. Use thumbs up or thumbs down if shown

## If something goes wrong

1. Check that your internet connection is working
2. Close and reopen the app
3. If sign-in fails, verify the redeem code with support
4. If the app still fails, export a support bundle and send it to the team

## Export support bundle

Run:
`.\export-support-bundle.ps1`

This collects local logs and deployment config files into a zip for support review.
